# git-hooks
По-умолчанию хуки лежат в папке проекта `.git/hooks`

Для смены пути к хукам используем команду
```sh
$ git config --global core.hooksPath ./git_hooks
```

###Пример с pre-commit
Хук выполняется перед коммитом, поэтому мы можем изменить файлы и добавить все изменения через `gid add`.

В рамках примера мы меняем данные в папке `./asssets`, если в этой папке произошли изменения.

Может пригодиться для запуска сборщиков js/css файлов.
